# Bank Loan Data Analysis Project

## Project Overview

This project analyzes bank loan data to understand loan applications, funding amounts, and repayment behavior. The goal is to identify key performance indicators (KPIs) and trends that help evaluate the bank's lending performance.

## Dataset

The dataset contains information about loan applications, funded amounts, interest rates, borrower details, and repayment data.

## Key Performance Indicators (KPIs)

* Total Loan Applications
* Total Funded Amount
* Total Amount Received
* Average Interest Rate
* Average Debt-to-Income Ratio (DTI)

## Good Loan vs Bad Loan Analysis

The project classifies loans into:

* **Good Loans**
* **Bad Loans**

Metrics analyzed:

* Good Loan Application Percentage
* Bad Loan Application Percentage
* Funded Amount
* Total Amount Received

## Visualizations

The project includes the following dashboards and charts:

* Monthly Trends by Issue Date
* Regional Analysis by State
* Loan Term Analysis
* Employee Length Analysis
* Loan Purpose Breakdown
* Home Ownership Analysis

## Tools Used

* Python
* Pandas
* Matplotlib / Seaborn
* Jupyter Notebook
* Excel

## Purpose

This analysis helps understand lending trends, borrower behavior, and loan performance, supporting better decision-making for financial institutions.
